# Academics 데이터 작성 가이드

이 폴더의 각 `.md` 파일은 하나의 학술 활동(논문, 발표, 학내 프로젝트 등)을 나타냅니다.

## Frontmatter 필드
- `title.ko` / `title.en`: 제목
- `description.ko` / `description.en`: 한 줄 설명
- `date`: `YYYY-MM-DD`
- `type`: `"conference"`, `"coursework"` 등
- `venue`: 학회명 또는 발표처
- `thumbnail`: `/public/images/academics/` 하위 이미지 경로
- `tech_stack`: 사용 기술/방법론 배열

## 본문 작성 규칙
`## 섹션명` (한국어) 다음에 `---`, 이어서 `## Section Name` (영어) 순서로 작성합니다.
