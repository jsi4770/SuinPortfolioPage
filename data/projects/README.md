# Projects 데이터 작성 가이드

이 폴더의 각 `.md` 파일은 하나의 프로젝트를 나타냅니다.

## Frontmatter 필드
- `title.ko` / `title.en`: 프로젝트 제목
- `description.ko` / `description.en`: 한 줄 설명
- `date`: `YYYY-MM-DD`
- `status`: `"진행 중"` 또는 `"완료"`
- `duration`: 진행 기간 텍스트
- `thumbnail`: `/public/images/projects/` 하위 이미지 경로
- `tech_stack`: 사용 기술 배열

## 본문 작성 규칙
`## 섹션명` (한국어) 다음에 `---`, 이어서 `## Section Name` (영어) 순서로 작성합니다.
